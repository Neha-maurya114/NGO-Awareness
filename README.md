# 🌟 NGO Awareness Webpage

A responsive NGO awareness website developed using **HTML** and **CSS** as part of my internship project with **InAmigos Foundation**.

## 📖 Project Overview

This project is designed to spread awareness about **InAmigos Foundation**, its mission, ongoing projects, and the positive impact it creates in society. The website provides visitors with information about the NGO and encourages them to volunteer, support, and contribute to its initiatives.

---

## ✨ Features

- 🏢 Introduction to InAmigos Foundation
- 📌 Ongoing NGO Projects
- 🌍 Social Impact & Mission
- 🖼️ Images and Visual Sections
- 🤝 Call-to-Action (Join Us, Volunteer, Support)
- 📱 Responsive Design
- 🎨 Clean and Modern User Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
